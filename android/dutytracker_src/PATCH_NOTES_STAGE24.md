# DutyTracker Android — Stage 24 (Actionable diagnostics)

## Что добавлено
- Единый сбор проблем "живучести" (Survivability): права, фон, уведомления, сеть, GPS, батарея, stale GPS, фильтр.
- Главный экран: список проблем с кнопками "Исправить" для каждого пункта.
- Экран Диагностики: те же проблемы + кнопки "Исправить" + быстрые переходы:
  - Уведомления
  - Интернет
  - Исключить из оптимизации батареи

## Где смотреть
- MainActivity: блок "Проблемы".
- DiagnosticsActivity: раздел "Проблемы" и блок "Быстрые настройки".

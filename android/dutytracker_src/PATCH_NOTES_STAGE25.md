# PATCH NOTES — Stage 25 (Smart Start)

## Что добавлено
1) Smart Start (предстартовая проверка) на главной кнопке "Включить трекинг":
   - при запуске трекинга показывается диалог со списком проблем (GPS/сеть/уведомления/оптимизация батареи/фоновые разрешения)
   - у каждой проблемы есть кнопка "Исправить" (переход в нужные настройки/запрос permission)
   - кнопка "Начать" позволяет включить трекинг даже если есть предупреждения

2) Offline-start:
   - если /api/tracker/start не удалось вызвать (нет сети/таймаут и т.п.), трекинг всё равно включается
   - точки копятся в очереди и будут отправлены позже; UploadWorker сам создаст сессию при первой успешной отправке

3) Auto-start after fix (best-effort):
   - если пользователь нажал Start, пошёл в настройки из диалога и вернулся, приложение попробует автоматически включить трекинг,
     когда устранены блокирующие причины (нет GPS-права/выключена геолокация/нет уведомлений)

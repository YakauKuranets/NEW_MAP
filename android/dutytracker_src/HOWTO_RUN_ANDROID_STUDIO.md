# DutyTracker Android (Quick Start)

1) Open THIS folder in Android Studio (File -> Open), the folder contains settings.gradle.kts and gradlew.bat.
2) Wait for Gradle Sync.
3) Run -> Edit Configurations -> + -> Android App -> Module: app (if config didn't appear automatically).
4) Start an emulator (Tools -> Device Manager) or connect a phone, then press Run.

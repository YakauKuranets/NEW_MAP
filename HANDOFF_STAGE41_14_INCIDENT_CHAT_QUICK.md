# Stage41.14 — Быстрый чат инцидента из Command Center

Цель: убрать лишние клики и сделать «Инцидент → Чат» максимально прямым.

## Что сделано

1) **Кнопка «Чат» внутри карточки инцидента (drawer / Overview)**
- В overview инцидента добавлена кнопка **«Чат»**.
- Кнопка использует ту же логику что и верхняя иконка чата в drawer (если она доступна).
- Fallback: если по каким-то причинам верхняя кнопка недоступна — пытаемся открыть чат через `chat2OpenForIncident()`.

2) **Кнопка чата в списке «Инциденты» (левая панель Command Center)**
- В каждой строке списка добавлена кнопка с иконкой чата.
- Нажатие открывает чат инцидента через `chat2OpenForIncident(id)`.
- Fallback: если глобальной функции нет — делаем `ensure_incident_channel` и открываем канал через `chat2OpenChannel()`.

## Файлы
- `static/js/admin_panel.js`
  - `_renderIncidentOverview()` — добавлена кнопка «Чат»
  - `openIncidentCard()` — wiring обработчика для кнопки «Чат»
- `static/js/incidents_panel_cc.js`
  - добавлена action-кнопка `data-act="chat"` + обработчик

## Как проверить (5 шагов)

1) Открой Command Center → вкладка **Incidents**.
2) Нажми иконку **чата** у любого инцидента — должен открыться Chat2 с заголовком `Инцидент #ID`.
3) Открой карточку инцидента (drawer) → вкладка Overview.
4) Нажми кнопку **«Чат»** — должен открыться тот же чат.
5) Напиши тестовое сообщение — после обновления список инцидентов должен корректно показать unread/обновления (если сообщение писал другой пользователь).
